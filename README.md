## 萌是什么？
萌 (日语：萌え，英语：Moe) 是起源于日本ACG次文化的常用语之一，也是萌文化时代最具代表性的用语。用于形容对ACG作品中的虚拟人物产生的一种强烈的、类似于恋爱的情感。如今逐渐成为“可爱”的同义词。
（引用自萌娘百科(https://zh.moegirl.org/%E8%90%8C#.E8.90.8C.E5.8C.96)）
